# Challenges, Achievements, and Badges System

## Overview
This document describes how challenges, achievements, and badges encourage users to practice and engage in the community.

## Challenge Definition
A challenge is a time-based activity users can join for free.

Example:
- Name: 7-Day Practice Challenge
- Requirement: Upload one practice clip per day
- Duration: 7 days

## Achievement Definition
An achievement is a record that a user has successfully completed a goal.

Examples:
- Completed 7-Day Practice Challenge
- Won a Monthly Championship
- Completed First Paid Lesson as Tutor

## Badge Definition
A badge is a visual reward associated with specific achievements.
Not every achievement grants a badge.

Examples:
- Consistent Performer (earned from 7-Day Practice Challenge)
- Top Mentor (earned from tutoring milestones)

## Relationship Between Concepts
- Completing a challenge creates an achievement.
- Some achievements grant a badge.
- Badges are optional to display on a userâ€™s profile.

## Progress Tracking
For each user:
- Track active challenges
- Track completed achievements
- Track earned badges
- Track which badges are displayed publicly

## Profile Display Rules
- Users can choose which badges to display.
- A profile shows:
  - Total number of achievements earned
  - Selected badges chosen by the user

## Completion Rules
A challenge is completed when:
- One valid submission is uploaded per day
- All required days are completed within the challenge duration

## Failure Cases
- Missed required activity
- Late submission
- Deleted submission

## Notes
- Challenges are free to join.
- Achievements are permanent.
- Badges cannot be revoked but can be hidden.
